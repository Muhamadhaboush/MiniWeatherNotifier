# MiniWeatherNotifier

A simple Java console application that shows random weather information and provides basic advice based on the weather conditions.

## Features
- Randomly generates temperature between -10°C and 35°C
- Randomly selects weather conditions: Sunny, Cloudy, Rainy, Windy, Snowy
- Provides simple advice based on the weather (e.g., “Take an umbrella!”)
- Console-based, lightweight, beginner-friendly project for learning Java

## Technologies Used
- Java
- Scanner for input
- Random class for generating random weather and temperature
